# b站视频/动态评论转图片

#### 介绍
b站评论获取神器，此工具由可将Bilibili视频/动态评论区里的评论转换成PNG图片。应用场景如：呈现观众评论、评论互动、频道，将会陆续完善其他功能，敬请期待。

#### 功能特色
1. 无限滚动获取评论
2. 瀑布流展示评论内容
3. 自由选择需要下载的评论
4. 支持视频、文字/图片动态评论
5. ZIP 压缩包打包下载


#### 技术栈
+ vue
+ element ui
+ html2canvas
+ JSZip


#### 使用说明
```
npm i
npm run serve
```
